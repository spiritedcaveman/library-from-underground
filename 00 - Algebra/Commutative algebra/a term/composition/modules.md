
A left $R$-module $M$ is called cyclic if $M$ can be generated by a single element i.e. $M = (x) = Rx = \{rx \mid r \in R\}$ for some $x$ in $M$. Similarly, a right $R$-module $N$ is cyclic if $N = yR$ for some $y \in N$.

$M \to R / Ann_Rx$
